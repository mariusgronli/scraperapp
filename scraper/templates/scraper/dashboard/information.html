Journey AS - Real estate algorithm




Collects all the data for you

The app scans market listings in Norway to collect data about listings. The data is collected daily and updated at 8pm every day.


Python analyzes your data

The app uses powerfull python libraries like pandas, numpy, scipy, math and custom made algorithims to collect and format data.


Presents the analyzed data

All analyzed data are displayed in a very simplistic format. Which makes it easy for you to capture key data in an instant. You will get key statistics about your hosuing markets. As well as formated data which can be used in powerful analysis.
